# Dotfiles
Configuration files for:
- git
- powerlevel10k
- tmux
- vim
- zsh
- fish
- neovim
- latexindent
- rime
- visual studio code
- ghostty
- bat
- pip
- ruff
- uv
- starship
- zellij

install dotbot via `uv` or `homebrew`, then run
```shell
dotbot -d ~/.dotfiles -c ~/.dotfiles/install.conf.yaml
```
